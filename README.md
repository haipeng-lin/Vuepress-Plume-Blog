## blog

### 介绍

网站基于 [vuepress](https://vuepress.vuejs.org/) 和 [vuepress-theme-plume](https://github.com/pengzhanbo/vuepress-theme-plume) 构建生成

- 预览地址：[Flash](https://haipeng-lin.cn/)


### 安装教程

```shell
# 启动开发服务
pnpm docs:dev

# 构建生产包
pnpm docs:build

# 本地预览生产服务
pnpm docs:preview

# 更新 vuepress 和主题
pnpm vp-update
```
