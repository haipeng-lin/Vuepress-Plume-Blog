# Vuepress-Plume-Blog

## 介绍

预览地址：[Flash](https://haipeng-lin.cn/)

本站基于 Vuepress-Plume 主题构建，并进行美化，新特性如下：

- 自定义首页
- 自定义 "关于" 页
- 足迹，集成高德地图 API
- 藏宝阁

## 使用教程

````shell
# clone 代码
git clone git@gitee.com:linhaipengg/Vuepress-Plume-Blog.git

# 安装依赖
npm install

# 运行
npm run docs:dev

# 部署
npm run docs:build
````
