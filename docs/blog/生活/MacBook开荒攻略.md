## 1.常用快捷键

由于 mac 的键盘和 win 的键盘位置不一样，已经习惯了 win 键盘操作的我，将外接键盘的 ctrl 键修饰为 command 键，以及将 command 键修饰为 ctrl 键；将内置键盘的 fn 键修饰为 command 键，以及将 command 键修饰为 fn 键

|                | 原 Mac                      | 外接键盘              | 内置键盘          |
| -------------- | --------------------------- | --------------------- | ----------------- |
| 移动文件       | command+c、command+option+v | ctrl+c、ctrl+option+v | fn+c、fn+option+v |
| 光标移动到行头 | ctrl+a                      | command+a             | ctrl+a            |
| 光标移动到行尾 | ctrl+e                      | command+e             | ctrl+e            |
| 锁屏           | command+ctrl+q              | ctrl+command+q        | fn+ctrl+q         |
