## 前言

本篇用于记录 Linux 的常用命令

## 目录文件

1. 创建、删除目录：mkdir、rmdir

2. 文件：

	1. 创建、删除文件：touch、rm
	2. 查看文件：cat

3. 移动文件/目录、重命名：mv

	```sh
	# 将 file.txt 移动到 /home/user/Documents/ 目录
	mv file.txt /home/user/Documents/
	
	# 将 oldfile.txt 重命名为 newfile.txt
	mv oldfile.txt newfile.txt
	```

4. 复制文件、目录：cp

	```sh
	# 将 file1.txt 复制为 file2.txt
	cp file1.txt file2.txt
	
	# 递归复制目录
	cp -r dir1/ dir2/
	```

5. 查找文件指定行

	1. tail -n -k 文件名：查找文件的最后k行
	2. tail -n +k 文件名：查找文件的第k行到最后一行
	3. head -n k 文件名：查找文件的前k行
	4. sed -n '6p' a.txt：查看文件的第6行

6. 文件搜索指定文本：grep 指定文本 文件路径

## 权限

1. ls -l：查看文件或目录的详细信息，包括权限信息

	```sh
	ls -l file.txt
	```

2. chmod：修改文件或目录的权限

	```sh
	# 改变文件权限为 rwxr-xr--
	chmod 755 file.txt
	
	# 通过符号方式改变权限
	chmod u+x file.txt    # 给文件的所有者添加执行权限
	chmod g-w file.txt    # 给文件的所属组移除写权限
	chmod o+r file.txt    # 给其他用户添加读取权限
	```

3. chown：变文件或目录的所有者和/或所属组

	```sh
	chown user:group file.txt     # 改变文件的所有者和组
	chown user file.txt           # 仅改变文件的所有者
	chown :group file.txt         # 仅改变文件的所属组
	```

	
## 防火墙

查看防火墙状态：

```shell
systemctl status firewalld 
```

开启防火墙：

```shell
systemctl start firewalld.service
```

关闭防火墙：

```shell
systemctl stop firewalld.service
```

开启开机自启动：

```shell
systemctl enable firewalld.service
```

关闭开机自启动：

```shell
systemctl disable firewalld.service
```

查询开放端口：

```shell
firewall-cmd --list-port
```

开放指定端口：

```shell
firewall-cmd --zone=public --add-port=15672/tcp --permanent
```

移除指定端口：

```shell
firewall-cmd --zone=public --remove-port=80/tcp --permanent
```

重启防火墙：

```shell
firewall-cmd --reload
```

## 端口

`lsof` (List Open Files) ：查看特定端口被哪个进程占用

```shell
# 查看 80 端口的占用情况
lsof -i :80
```

kill：杀死指定进程号的进程

```shell
# 杀死进程号30000的进程
kill 30000
```
