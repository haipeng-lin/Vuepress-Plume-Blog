## 1.介绍

一个内容分享平台，实现了文章发布和检索、点赞收藏、文章评论、用户排行榜以及知识AI等功能

- 演示地址：[知识流前台](http://8.134.56.162/)
- 代码地址：[Github](https://github.com/haipeng-lin/knowledge-stream)

## 2.技术栈

- 后端：SpringBoot + MySQL + Redis + ES + Cannal
- 前端：Vue
- 第三方应用接口：
  - 接收用户发送给微信公众号的消息
  - 讯飞星火大模型 API
  - 通义千问 API
  - 智谱 API

## 3.模块功能

- 系统模块
  - 登录注册，基于微信公众号+验证码的登录
  - 用户活跃排行榜
  - 用户白名单
- 文章模块
  - 发布文章、ES 检索文章
  - 点赞收藏文章
  - 评论文章及回复
- AI 模块：
  - 基于策略模式接入多个大模型 AI
- 后台模块：
  - 标签管理
  - 文章管理
  - 教程管理
